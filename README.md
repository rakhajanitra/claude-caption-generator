# 🏡 Claude Caption Generator

An AI-powered caption generator built using Claude API by Anthropic.  
This tool helps create engaging real estate captions and hashtags for marketing content.

---

## 🚀 Features

- Generate high-converting property captions
- Automatically create relevant hashtags
- Simple and lightweight Node.js setup
- Powered by Claude AI

---

## 🛠️ Tech Stack

- Node.js
- Claude API (Anthropic)
